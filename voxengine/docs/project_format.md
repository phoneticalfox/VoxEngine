# VoxEngine Project Format (Draft)

A project is a directory.

## Required

- `project.json`
- `cast/`
- `script/`
- `renders/`

## Suggested layout

```
MyProject/
  project.json
  cast/
    actor_name/
      consent.json
      reference.wav
      embedding.bin
  script/
    scenes.json
  renders/
    scene01/
      line_001.wav
```
